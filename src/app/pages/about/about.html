ABOUT THE PROJECT

What is EduTrack and Why Was It Built?

EduTrack is a single-page application (SPA) built with Angular 17+ as a capstone project for an Application Development course. It demonstrates a full range of Angular fundamentals — component architecture, routing with guards, reactive forms, signals, HTTP client integration, and RxJS observables — while delivering a genuinely useful tool for students and self-learners.

The app connects to the Open Library REST API to surface real educational resources, giving users a live, searchable catalog of books aligned to their subjects and goals. Every feature was designed with SDG 4 — Quality Education — in mind.

SDG 4 Connection

EduTrack directly supports access to quality educational resources, promotes self-directed lifelong learning, and tracks academic progress — all core pillars of the UN's 2030 education agenda.

Course Context

Built as a final submission for Application Development 1, demonstrating mastery of Angular standalone components, routing, HTTP, and reactive state.

FEATURES

What EduTrack Can Do

01

Auth with Route Guards

Login/logout flow using Angular signals and localStorage. A canActivate guard protects the Dashboard, and a canDeactivate guard prevents accidental navigation away from unsaved changes.

02

Live Book Search via HTTP

ResourceService uses Angular's HttpClient to query the Open Library REST API in real time. Results are transformed via RxJS map() and fall back to hardcoded data via catchError() on failure.

03

Goal & Subject Tracking

Dashboard features subject cards, academic goal lists, a Pomodoro focus timer, and an achievement system — all powered by Angular signals for reactive state updates.

04

SDG 4 Info Page

A dedicated page covering SDG 4 targets, global education statistics, an interactive quiz, and EduTrack's direct contribution to the UN's 2030 education agenda.
